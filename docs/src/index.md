# CurveDiagrams

A package for storing and manipulating the anyonic fusion basis used to
represent the quantum state of a lattice model realization of an anyon theory.

For clarity, this package does not facilitate storing or manipulating quantum
**states**, only the basis used to represent those **states**. For the former,
see e.g. [TensorKit.jl](https://quantumkithub.github.io/TensorKit.jl/stable/).
